As Flutter developer chagne app to respect one to many relationship between category and product:
- One category can have multiple products
- All names have to be unique across the same category. Names can be duplicated in different categories. 
- Name can be deleted only if products with that name doesn't exist
- Category can be deleted only if products with that category doesn't exist
- During adding Name user have to choose category it belongs or add new one
- During adding a product user can choose Names only from choosen Category in previous step
